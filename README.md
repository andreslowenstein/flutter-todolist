# flutter-todolist
A simple todo list app made with Flutter and Firebase cloud

developed by Andrés Lowenstein

### Install:
```sh
flutter pub get
```
### Run:
```sh
flutter build apk
```
Or you can run the app in your Android/IOS emulator directly.
